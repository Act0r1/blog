Добро пожаловать в первый пост моего нового блога! 

Если вы читаете это, значит всё работает как надо. Блог построен на [Astro](https://astro.build/) — потрясающем фреймворке для контентно-ориентированных сайтов, который генерирует статичный HTML, отправляя минимум JavaScript на клиент (или вообще ноль, как в нашем случае).

В качестве рантайма и пакетного менеджера используется **Bun**. Он невероятно быстрый!

### Зачем нужен еще один блог?

Иногда хочется просто сесть и написать пару мыслей, не отвлекаясь на сложный дизайн, всплывающие окна и тонны рекламы. 
Этот минималистичный дизайн создан специально для того, чтобы сфокусироваться на главном — **тексте**.

Вы могли заметить выпадающий список сверху. Это переключатель шрифтов! Попробуйте поменять системный шрифт на `Inter`, `Roboto`, `Monospace` или `Georgia`, чтобы найти тот, который читать приятнее всего.

```javascript
// Простой пример кода, чтобы проверить подсветку (если добавим)
const greet = (name) => {
  console.log(`Привет, ${name}!`);
};

greet("мир");
```
